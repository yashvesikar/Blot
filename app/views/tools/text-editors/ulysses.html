Ulysses

Write your posts with Ulysses

Blot supports the TextBundle format, which lets you embed images in Markdown files. We recommend configuring Ulysses to use this file format.

Configure Ulysses on iOS

  1. Create a new 'External folders' if you haven't already. Then:
  2. hard press your Dropbox folder to open the settings menu
  3. press 'Get Info' then make sure 'Read and Write Markdown files' is enabled, and set 'Default File Extension' to '.textbundle'.
